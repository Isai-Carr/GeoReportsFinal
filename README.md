# GeoReport PWA

Aplicación web progresiva para reportar incidencias con fotografía y ubicación.

## Funcionalidades
- Crear reportes con foto
- Obtener ubicación GPS
- Guardado en IndexedDB
- Funcionamiento offline
- Notificaciones

## Tecnologías
- HTML, CSS, JavaScript
- Service Worker
- IndexedDB
- APIs del navegador

## Características PWA
- Instalación en dispositivo
- Funcionamiento sin conexión
- Cache de recursos
